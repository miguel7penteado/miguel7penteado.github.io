# Boston Open Data Hub

## Sobre o Boston Open Data Hub
[Analyze Boston](https://data.boston.gov) é o centro de dados aberto da cidade de Boston para encontrar fatos, números e mapas relacionados às nossas vidas dentro da cidade. Estamos trabalhando para tornar esta a plataforma de tecnologia padrão para apoiar a publicação de informações públicas da cidade, na forma de dados, e para tornar essas informações fáceis de encontrar, acessar e usar por um público amplo. Esta plataforma é gerenciada pela equipe de análise da cidade.

O lançamento do Analyze Boston marca o ponto culminante de nossos esforços para fazer a transição para uma nova plataforma de dados abertos. Continuaremos nosso trabalho para selecionar conjuntos de dados atualizados e de alta qualidade e desenvolver uma plataforma amplamente acessível. Se você tiver comentários, preencha este formulário simples.

O Analyze Boston é possível graças ao financiamento concedido pelo Knight News Challenge on Libraries da Knight Foundation, que estabeleceu o projeto Open Data to Open Knowledge.
